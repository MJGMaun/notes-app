### Features

- Add note
- Read note
- Remove note
- List notes

###Installation

------------

`$ npm install `

###Commands

------------



| Command | Description                    |
| ------------- | ------------------------------ |
| `node app add --title="title here" --body="body here"`      | Add note          |
| `node app read --title="title here"`                                       | Read note        |
| `node app remove --title="title here"`                                  | Remove note   |
| `node app list`                                                                        | List notes         |

###End
